Student Registration System
- This project is a Student Registration System that uses a REST API and multiple clients to perform CRUD operations (Create, Read, Update, Delete).

It allows users to:
Register new users
View all registered users
Update user passwords
Delete users

Technologies Used
* Backend (REST API)
PHP
JSON (data storage)
XAMPP (Apache server)
* Clients
C# Windows Forms Application (Visual Studio)
C++ Console Application
Python Client
